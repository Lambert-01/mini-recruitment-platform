## Introduction
This is a mini recruitment platform built as a capstone project to demonstrate comprehensive software development skills. It includes user authentication, job listings, application submission, a basic job recommendation system, and an admin dashboard.

## Features
- **User Authentication**: Signup, login, and profile management.
- **Job Listings**: Display job listings with filtering and search.
- **Application Submission**: Multi-step form for submitting job applications.
- **Job Recommendation System**: Basic recommendation system based on user preferences.
- **Admin Dashboard**: Manage jobs and applications.

## Technologies Used
- **Frontend**: React, React Router, Axios
- **Backend**: Node.js, Express, MongoDB
- **Deployment**: Heroku (backend), Vercel (frontend)
